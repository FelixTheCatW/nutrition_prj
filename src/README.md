# Проект «Nutrition Tracker» (учебный)

[![Python](https://img.shields.io/badge/Python-3.9%2B-blue)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

**Программа для анализа питания и отслеживания потребления калорий, белков, жиров и углеводов.**  
Проект выполнен в рамках учебного курса по основам программирования.  
Реализованы два интерфейса: **консольный (curses)** и **Telegram‑бот**.

---

## 📦 Функциональные возможности

- Генерация случайных данных о пользователях и их приёмах пищи (реалистичные блюда разных кухонь мира).
- Расчёт индивидуальных норм (BMR, TDEE, целевая калорийность, БЖУ).
- 10 аналитических отчётов:
  1. **Персональная статистика** – среднее потребление, сравнение с целью, динамика по дням + график.
  2. **Анализ макронутриентов** – соотношение БЖУ в калориях + круговая диаграмма.
  3. **Топ самых частых блюд** – рейтинг по количеству приёмов.
  4. **Топ самых калорийных блюд** – средняя калорийность за приём.
  5. **Сравнение пользователей** – ИМТ, отклонение от цели, гистограмма.
  6. **Анализ приёмов по времени** – средняя калорийность завтрака/обеда/ужина/перекуса, поздние перекусы + столбчатая диаграмма.
  7. **Календарь питания** – калорийность по дням месяца, цветовая индикация + гистограмма.
  8. **Прогресс к цели** – расчёт накопленного дефицита/профицита, прогноз изменения веса + график накопленного дефицита.
  9. **Общая статистика** – количество приёмов, уникальных блюд, распределение целей.
  10. **Отчёт по эффективности** – дни с нормой ±10%, дни с пропуском приёмов, гистограмма.

- **Графики** (`matplotlib`): линейные, круговые, столбчатые, с накоплением – открываются в отдельных окнах (для curses) или отправляются в Telegram.
- **Текстовые таблицы** форматируются с помощью `tabulate` и поддерживают русские заголовки.

## 📥 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/FelixTheCatW/nutration_prj.git
cd nutration_prj
```

### 2. Виртуальное окружение (рекомендуется)
```bash
python -m venv .venv
source .venv/bin/activate      # Linux / macOS
.venv\Scripts\activate          # Windows
```
### 3. Установка зависимостей
```bash
pip install pandas matplotlib tabulate faker python-telegram-bot
```
Либо создайте requirements.txt с указанными библиотеками и выполните pip install -r requirements.txt.
### 4. Генерация данных (если нет CSV)
```bash
python generate_data.py --users 10 --days 30 --output data/nutrition_data.csv
```
### 5. Запуск приложения
🖥 Консольный (curses)
```bash
python app.py
```
    Управление: ↑/↓ – выбор отчёта, Enter – запуск, U – смена пользователя, Q – выход.

    Графики открываются в отдельных окнах (не блокируют ввод).

### 🤖 Telegram‑бот

    Получить токен у @BotFather.

    В файле bot.py заменить YOUR_BOT_TOKEN_HERE на свой токен.

    Запустить:

```bash
python -m src.tg.bot
```
    В Telegram: /start → выбор пользователя → выбор отчёта → бот пришлёт текст и (если есть) изображение графика.
